 * @callback wdcw~connectorSetup

 * @param {string} phase

 *   in. One of: auth, interactive, or gatherData.
 *
 * @return {Promise}
 *
 * @example <caption>Basic auth phase example</caption>

 *   if (phase === 'auth') {

 * }
